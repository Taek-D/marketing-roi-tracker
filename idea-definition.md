# 💡 마케팅 ROI 최적화 프로젝트 - 아이디어 정의 (STEP 1)

## 🎯 프로젝트 한 줄 정의
**"중소 이커머스를 위한 무료 멀티채널 마케팅 ROI 대시보드: 30초 셋업으로 Google Ads, Facebook Ads, 네이버 GFA 데이터를 자동 수집하고 채널별 기여도를 시각화하는 스프레드시트 기반 솔루션"**

---

## 🚀 내가 만들고 싶은 것

### 핵심 개념
> "마케팅 예산이 제한적인 스타트업/중소 이커머스가 **Rockerbox 같은 고가 솔루션 없이도** 채널별 ROI를 명확히 파악하고 데이터 기반 예산 재배치 결정을 할 수 있게 돕는 도구"

**대상**: 월 광고비 $5K~$30K 규모의 이커머스/DTC 브랜드  
**형태**: Google Sheets + Apps Script 기반 자동화 대시보드  
**가격**: 100% 무료 (오픈소스)  
**셋업 시간**: 30초 (API 키만 입력)

---

## 🎯 핵심 기능 (Core Features)

### 🔹 핵심 기능 1: **멀티채널 데이터 자동 수집**
- **기능 설명**:
  - Google Ads, Facebook Ads, 네이버 GFA, 카카오모먼트, TikTok Ads API 연동
  - 일별/주별/월별 광고비, 노출수, 클릭수, 전환수, 매출 자동 수집
  - Google Sheets에 원본 데이터 저장 (Historical Data 축적)
- **자동화 방식**:
  - Apps Script로 매일 오전 9시 자동 실행
  - API 에러 발생 시 Slack/Discord 알림
- **기대 효과**:
  - 기존 수동 작업 (30분/일) → 0초
  - 데이터 정합성 향상 (실수 없음)

---

### 🔹 핵심 기능 2: **멀티터치 애트리뷰션 자동 계산**
- **기능 설명**:
  - **4가지 애트리뷰션 모델** 지원:
    1. **First-Touch**: 첫 번째 접촉 채널에 100% 크레딧
    2. **Last-Touch**: 마지막 접촉 채널에 100% 크레딧
    3. **Linear**: 모든 터치포인트에 균등 분배
    4. **Time-Decay**: 시간에 따라 가중치 (최근 터치가 높은 가중치)
  - GA4 User Journey 데이터와 광고 플랫폼 데이터 결합
  - 채널별 기여 매출(Attributed Revenue) 자동 계산
- **시각화**:
  - 채널별 기여도 파이 차트
  - 모델별 비교 테이블 (같은 전환에 대해 4가지 모델 결과 비교)
- **기대 효과**:
  - "Facebook Ads가 First-Touch에서는 30%지만 Last-Touch에서는 10%" 같은 인사이트 도출
  - 채널별 역할(인지 vs 전환) 명확화

---

### 🔹 핵심 기능 3: **ROI & ROAS 실시간 대시보드**
- **기능 설명**:
  - 주요 지표 자동 계산:
    - **ROAS** (Return on Ad Spend) = 광고 매출 / 광고비
    - **ROI** = (매출 - 광고비 - 제품 원가) / 광고비
    - **CPA** (Cost Per Acquisition) = 광고비 / 전환수
    - **LTV/CAC 비율** (선택적)
  - 채널별, 캠페인별, 광고소재별 ROI 비교
  - "목표 ROAS 3 이상" 같은 가드레일 자동 체크
- **시각화**:
  - Heatmap (채널×주차별 ROAS)
  - Waterfall Chart (매출 구성 분해)
  - Sparkline (주간 트렌드)
- **기대 효과**:
  - "Google Ads의 ROAS는 4.2인데 Facebook Ads는 1.8" → 예산 재배치 근거 확보
  - 주간 리뷰 미팅에서 5분 내 핵심 지표 공유

---

### 🔹 핵심 기능 4 (보너스): **예산 시뮬레이터**
- **기능 설명**:
  - "만약 Facebook Ads 예산을 20% 줄이고 Google Ads에 20% 더 투자하면?"
  - 과거 3개월 데이터 기반 시뮬레이션 (단순 선형 가정)
  - 예상 ROAS, 예상 매출 증가분 계산
- **입력**:
  - 채널별 예산 조정 비율 (슬라이더)
- **출력**:
  - 조정 후 예상 ROAS, 예상 ROI
  - 시뮬레이션 결과 요약 문장 (예: "총 매출 +$5,200 증가 예상")
- **기대 효과**:
  - 예산 재배치 결정 전 사전 검증
  - 경영진 설득 자료로 활용

---

## ⚠️ 기존 솔루션의 문제점

### 문제점 1: **Rockerbox/Northbeam은 너무 비싸서 불편해**
- **현재 상황**: 연 $50K~$100K (월 $5K 이상)
- **문제**: 스타트업은 마케팅 예산보다 도구 비용이 더 클 수 있음
- **우리 솔루션**: **100% 무료** (Google Sheets 무료 플랜 활용)

---

### 문제점 2: **Amplitude/Mixpanel은 프로덕트 분석 위주라 마케팅 ROI 측정이 약해**
- **현재 상황**: 사용자 행동 분석은 강하지만 광고비-매출 연결 약함
- **문제**: 광고 플랫폼과 직접 통합 부족, UTM 기반 애트리뷰션만 지원
- **우리 솔루션**: **광고 플랫폼 API 직접 연동** + GA4 User Journey 결합

---

### 문제점 3: **GA4는 무료지만 멀티터치 애트리뷰션 설정이 복잡해**
- **현재 상황**: 
  - GA4의 Data-Driven Attribution은 "블랙박스" (설명 불가)
  - 커스텀 애트리뷰션 모델 직접 구현 어려움
- **문제**: 비전문가는 사용하기 어려움
- **우리 솔루션**: **4가지 애트리뷰션 모델 원클릭 전환** + 모델별 비교 테이블 제공

---

### 문제점 4: **수동 집계는 시간 낭비 + 실수 발생**
- **현재 상황**: 
  - 마케터가 매일 아침 Facebook Ads Manager, Google Ads, 네이버 광고관리자 들어가서 수동 복사
  - 엑셀에서 VLOOKUP, SUMIF로 수동 계산
- **문제**: 일일 30분 소요, 데이터 실수 빈번
- **우리 솔루션**: **자동화 (Apps Script)** + 일일 정합성 체크

---

## ✨ 우리 솔루션만의 차별점

### 차별점 1: **30초 셋업 (Zero-Code)**
- 사용자는 API 키만 입력하면 끝 (코딩 불필요)
- "템플릿 복사 → API 키 입력 → 실행" 3단계

---

### 차별점 2: **한국 시장 특화**
- 네이버 GFA, 카카오모먼트 기본 지원
- 한국어 UI/대시보드
- 한국 이커머스 KPI (네이버 쇼핑 검색광고, 카카오 선물하기 등)

---

### 차별점 3: **투명한 애트리뷰션 로직**
- 각 모델의 계산 로직을 시트에 공개
- "왜 이 채널이 20% 기여했는지" 설명 가능
- SHAP Value 같은 복잡한 ML 없이 직관적 이해

---

### 차별점 4: **오픈소스 & 커뮤니티**
- GitHub에 공개 → 사용자 피드백 반영
- 템플릿 갤러리 (업종별 템플릿: 패션, 뷰티, 식품 등)
- 슬랙 커뮤니티에서 질문/답변

## 🔍 솔직한 평가 & 피드백

### ✅ **이 아이디어가 가치 있는 이유**

1. **명확한 페인 포인트 해결**:
   - "비싼 도구 vs 수동 작업" 사이의 공백 메움
   - 실제 중소 이커머스의 80%는 여전히 엑셀 사용 중

2. **검증된 시장**:
   - 마케팅 애널리틱스 시장 연 12% 성장 중
   - GA4의 복잡성 + Amplitude의 높은 가격 → 중간 솔루션 수요 높음

3. **빠른 MVP 가능**:
   - Google Sheets + Apps Script → 인프라 비용 0원
   - 2주 내 기본 버전 완성 가능

4. **포트폴리오 차별화**:
   - 대부분의 지원자는 "분석" 프로젝트만 함
   - 이 프로젝트는 "분석 + 자동화 + 제품화" 전 영역 커버

---

### ⚠️ **잠재적 리스크**

1. **API 유지보수 비용**:
   - Facebook/Google Ads API는 자주 변경됨
   - 장기 유지보수 부담

2. **스케일 한계**:
   - Google Sheets는 1만 행 이상에서 느려짐
   - 대규모 기업은 사용 불가 (타겟 아님)

3. **수익 모델 부재**:
   - 무료 오픈소스는 좋지만, 실제 창업으로 이어지기 어려움
   - 포트폴리오용으로만 가치 있음

## 📦 MVP (최소 기능 제품)에 꼭 필요한 3가지

### ✅ MVP 기능 1: **2개 채널 자동 수집** (Google Ads + Facebook Ads)
- 이유: 대부분의 이커머스가 이 두 채널 사용
- 범위: 일별 광고비, 클릭수, 전환수, 매출만 수집 (상세 지표 제외)

---

### ✅ MVP 기능 2: **Last-Touch 애트리뷰션** (단일 모델)
- 이유: 가장 직관적이고 구현 쉬움
- 범위: 4가지 모델은 v2에서 추가 (MVP는 단순화)

---

### ✅ MVP 기능 3: **ROAS 대시보드** (채널별 비교)
- 이유: 가장 많이 보는 지표
- 범위: 파이 차트 1개 + 테이블 1개만
