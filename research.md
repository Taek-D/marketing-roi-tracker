# 📊 마케팅 ROI 최적화 프로젝트 - 시장 조사 (STEP 0)

## 🎯 조사 목적
데이터 분석가 지원 시 차별화 포인트가 될 수 있는 **마케팅 ROI 최적화 프로젝트**의 시장 현황을 파악하고, 포트폴리오에 적용할 핵심 인사이트를 도출합니다.

---

## 1️⃣ 마케팅 애트리뷰션 & ROI 최적화 분야에서 성공한 5가지 서비스

### 🏆 1. **Cometly** - 멀티터치 애트리뷰션 전문
- **핵심 강점**: 서버 사이드 트래킹으로 iOS 14+ 프라이버시 제약 극복
- **차별점**: 광고 플랫폼(Facebook/Google Ads)과 네이티브 통합으로 픽셀 손실 없이 전환 추적
- **타겟**: 퍼포먼스 마케터, 이커머스 브랜드
- **프라이싱**: 스타터 $149/월 → 엔터프라이즈 $999+/월
- **핵심 기능**: 
  - 퍼스트파티 데이터 기반 애트리뷰션
  - 크로스 디바이스 트래킹
  - Shopify/WooCommerce 직접 통합
- **출처**: [Cometly 공식 사이트](https://www.cometly.com/)

---

### 🏆 2. **Mixpanel** - 제품 분석 + 행동 코호트
- **핵심 강점**: 사용자 행동 데이터 기반의 깊이 있는 코호트 분석
- **차별점**: 
  - 이벤트 기반 트래킹으로 사용자 여정(Journey) 전체 가시화
  - 퍼널 분석, 리텐션 대시보드 기본 제공
  - 실시간 세그먼트 생성 및 타겟팅 자동화
- **타겟**: SaaS, 모바일 앱, 제품 중심 기업
- **프라이싱**: 프리 플랜 → Growth $28/월 → Enterprise 커스텀
- **핵심 기능**:
  - 행동 기반 세그먼트 (Behavioral Cohorts)
  - A/B 테스트 통합
  - 예측 분석 (Churn 예측)
- **출처**: [Mixpanel 공식 사이트](https://mixpanel.com/)

---

### 🏆 3. **Amplitude** - 제품 분석 플랫폼
- **핵심 강점**: 마케팅 캠페인의 전환율(Conversion Rate) 최적화에 특화
- **차별점**:
  - 사용자 행동 분석 + 퍼포먼스 마케팅 ROI 측정 통합
  - 무료 플랜에서도 월 10만 이벤트 무료 제공
  - 세션 리플레이(Session Replay) 제공
- **타겟**: 데이터 중심 마케터, 제품 팀, Growth 팀
- **프라이싱**: Free → Plus $61/월 → Growth $995/월
- **핵심 기능**:
  - 사용자 여정 시각화 (Path Analysis)
  - 리텐션 코호트, 퍼널, Impact 분석
  - 타겟팅 자동화
- **출처**: [Amplitude 공식 사이트](https://amplitude.com/solutions/marketing)

---

### 🏆 4. **Rockerbox** - 엔터프라이즈 마케팅 측정 플랫폼
- **핵심 강점**: 멀티터치 애트리뷰션(MTA) + 마케팅 믹스 모델링(MMM) + 인크리멘탈리티 테스트 통합
- **차별점**:
  - TV, 라디오, 오프라인 광고까지 통합 측정
  - 대규모 브랜드에 특화 (엔터프라이즈급)
  - 실시간 대시보드 + 자동화된 보고서
- **타겟**: 대규모 이커머스, DTC 브랜드, 엔터프라이즈
- **프라이싱**: 커스텀 (연 $50K~)
- **핵심 기능**:
  - 채널별 기여도 분석 (Channel Attribution)
  - 예산 최적화 시뮬레이션
  - 인크리멘탈리티(Incrementality) 측정
- **출처**: [Rockerbox 공식 사이트](https://www.rockerbox.com/)

---

### 🏆 5. **Google Analytics 4 (GA4)** - 무료 + 강력한 기본 도구
- **핵심 강점**: 무료이면서도 강력한 전환 트래킹 및 애트리뷰션
- **차별점**:
  - 이벤트 기반 데이터 모델 (페이지뷰 중심에서 벗어남)
  - BigQuery 무료 연동 (데이터 분석가에게 최적)
  - 머신러닝 기반 예측 지표 (Churn, Revenue 예측)
- **타겟**: 모든 규모의 비즈니스
- **프라이싱**: 100% 무료 (360 버전은 유료)
- **핵심 기능**:
  - 데이터 기반 애트리뷰션 (Data-Driven Attribution)
  - 사용자 경로 분석 (User Journey)
  - 맞춤 퍼널, 세그먼트, 탐색 보고서
- **출처**: [Google Analytics 4](https://analytics.google.com/)

## 2️⃣ 사용자들이 공통적으로 불만을 느끼는 점

### 😤 **문제점 1: 데이터 사일로 (Data Silos)**
- **상황**: 광고 플랫폼(Facebook Ads, Google Ads), CRM(HubSpot), GA4, Shopify 등이 분산되어 있음
- **문제**: 
  - 채널별 성과를 수동으로 취합해야 함
  - 실시간 통합 대시보드 부재
  - 마케팅 예산 재배치 의사결정이 늦어짐
- **빈도**: 사용자 리뷰 약 47%가 "데이터 통합이 어렵다"고 언급
- **출처**: [Growth-onomics: Multi-Touch Attribution Challenges](https://growth-onomics.com/multi-touch-attribution-models-challenges/)

---

### 😤 **문제점 2: 프라이버시 규제 (iOS 14+, GDPR, 쿠키 제한)**
- **상황**: 
  - iOS 14+ ATT(App Tracking Transparency) → Facebook Pixel 정확도 하락
  - GDPR, CCPA → 서드파티 쿠키 차단
  - 2024년부터 Chrome 쿠키 단계적 차단
- **문제**:
  - 멀티터치 애트리뷰션 모델 신뢰도 급락
  - 전환 추적 누락률 30~40%
  - 광고 플랫폼 리포트와 실제 매출 불일치
- **빈도**: 마케터의 약 60%가 "측정 정확도 저하"를 체감
- **출처**: [Braze: Marketing Attribution Challenges 2026](https://www.braze.com/resources/articles/challenges-of-marketing-attribution)

---

### 😤 **문제점 3: 애트리뷰션 모델의 복잡성**
- **상황**: 
  - 퍼스트터치, 라스트터치, 리니어, 타임디케이, 데이터 기반 등 모델이 너무 많음
  - 어떤 모델이 우리 비즈니스에 맞는지 모르겠다
- **문제**:
  - 모델 선택에 따라 채널별 기여도가 완전히 다르게 나옴
  - 경영진/마케팅 팀 간 신뢰 문제 발생
  - 복잡한 모델일수록 설명 가능성(Explainability) 부족
- **빈도**: 특히 중소기업(SMB)에서 "이해하기 어렵다" 언급 빈번
- **출처**: [Improvado: Marketing Attribution Models](https://improvado.io/blog/marketing-attribution-models)

---

### 😤 **문제점 4: 높은 비용 & 진입 장벽**
- **상황**: 
  - Rockerbox, Northbeam 같은 엔터프라이즈 솔루션은 연 $50K~$100K
  - Segment, mParticle 같은 CDP는 월 $1,000 이상
- **문제**:
  - 스타트업/중소기업은 비용 부담으로 도입 불가
  - 결국 엑셀/스프레드시트로 수동 집계
  - 데이터 파이프라인 구축에 엔지니어 리소스 필요
- **빈도**: SMB의 약 70%가 "비용이 너무 비싸다"고 응답
- **출처**: [LayerFive: Best Marketing Analytics Tools 2026](https://layerfive.com/blog/best-marketing-analytics-tools-2026-attribution/)

---

### 😤 **문제점 5: 오프라인-온라인 통합 부재**
- **상황**: 
  - 온라인 광고 → 오프라인 매장 방문 → 구매 전환 추적 불가
  - TV, 라디오, 옥외광고 등 전통 매체의 기여도 측정 어려움
- **문제**:
  - 옴니채널 브랜드는 ROI 전체 그림을 볼 수 없음
  - 오프라인 채널 예산을 줄이거나 늘려야 할지 판단 불가
- **빈도**: 리테일/패션 브랜드의 약 50%가 언급
- **출처**: [AdTaxi: Offline Attribution 2026](https://www.adtaxi.com/blog/unlocking-the-full-customer-journey-with-offline-attribution-in-2026/)

## 3️⃣ 아직 아무도 해결 못한 문제 (기회 영역)

### 💡 **문제 1: 실시간 예산 재배치 자동화**
- **현재 상황**: 
  - 대부분의 도구는 "리포팅"에 그침
  - 마케터가 수동으로 예산을 조정해야 함
- **미해결 영역**:
  - ROI가 낮은 채널에서 높은 채널로 **자동 예산 이동**
  - 실시간 CPM/CPC 변화에 따라 입찰가 조정 자동화
  - "만약 이 채널에 10% 더 투자하면?" 시뮬레이션 기능
- **왜 어려운가?**:
  - 광고 플랫폼 API 권한 제약
  - 예산 재배치 전략이 도메인별로 다름 (이커머스 vs SaaS vs 콘텐츠)
  - 실시간 데이터 파이프라인 구축 비용 높음

---

### 💡 **문제 2: 중소기업(SMB)을 위한 저비용 통합 솔루션**
- **현재 상황**:
  - 엔터프라이즈 솔루션은 너무 비쌈
  - GA4는 무료지만 멀티터치 애트리뷰션 약함
  - 중소기업은 여전히 엑셀로 수동 집계
- **미해결 영역**:
  - **월 $50 이하** 가격대의 통합 대시보드
  - 코드 없이(No-Code) 광고 플랫폼 데이터 자동 수집
  - 스프레드시트 베이스 + API 자동 연동 방식
- **왜 어려운가?**:
  - 수익 모델 확보 어려움 (SMB는 이탈률 높음)
  - 각 플랫폼(Facebook/Google/TikTok) API 유지보수 비용

---

### 💡 **문제 3: 설명 가능한 AI 기반 애트리뷰션**
- **현재 상황**:
  - 데이터 기반 애트리뷰션(Data-Driven Attribution)은 "블랙박스"
  - 경영진이 "왜 이 채널이 20% 기여했는지" 이해 못 함
- **미해결 영역**:
  - **Explainable AI (XAI)** 기반 애트리뷰션
  - "이 채널의 기여도가 높은 이유: 신규 유저 유입 90%, 평균 구매액 $120" 같은 상세 설명
  - SHAP, LIME 같은 기법을 마케팅 대시보드에 통합
- **왜 어려운가?**:
  - 마케터가 AI/ML 용어를 이해하기 어려움
  - 설명 가능성과 모델 정확도 간 트레이드오프

## 4️⃣ 진입하기 좋은 틈새시장 (Niche Opportunities)

### 🎯 **틈새 1: 스타트업/SMB용 "스프레드시트 기반 ROI 대시보드"**
- **타겟**: 월 광고비 $5K~$20K 규모의 스타트업, 이커머스
- **접근 방식**:
  - Google Sheets + Apps Script 기반 자동화
  - Facebook Ads API, Google Ads API로 데이터 자동 수집
  - 무료 또는 월 $20 이하 가격대
- **차별점**:
  - 코드 없이 "복사해서 바로 쓰는" 템플릿
  - 30초 셋업 (API 키만 입력하면 끝)
  - 오픈소스 커뮤니티 형성 가능
- **경쟁 우위**: Rockerbox 대비 1/100 가격, GA4 대비 높은 커스터마이징

---

### 🎯 **틈새 2: "인플루언서 마케팅" ROI 측정**
- **타겟**: DTC 브랜드, 뷰티/패션 이커머스
- **접근 방식**:
  - 인플루언서별 UTM 태깅 자동화
  - Instagram/TikTok/YouTube 게시물별 전환율 추적
  - CPM 대비 실제 매출 기여도 대시보드
- **차별점**:
  - 인플루언서 마케팅 전용 애트리뷰션 도구는 거의 없음
  - CreatorIQ, Aspire는 캠페인 관리 위주 (ROI 측정 약함)
- **경쟁 우위**: 인플루언서 마케팅 ROI의 "데이터 공백" 해결

---

### 🎯 **틈새 3: "리테일/오프라인" 옴니채널 애트리뷰션**
- **타겟**: 온라인+오프라인 매장을 동시 운영하는 브랜드 (예: 올리브영, 무신사 스토어)
- **접근 방식**:
  - 온라인 광고 노출 → 오프라인 매장 방문 → 구매 전환 추적
  - QR 코드, 쿠폰 코드, POS 데이터 통합
  - 지역별/매장별 ROI 분석
- **차별점**:
  - Rockerbox는 비싸고, GA4는 오프라인 추적 약함
  - 모바일 위치 데이터(Location Data) 활용 가능
- **경쟁 우위**: 한국 리테일 시장에 특화 (CU, GS25, 이마트 등)

---

### 🎯 **틈새 4: "구독 경제(Subscription)" 전용 LTV-CAC 최적화**
- **타겟**: SaaS, OTT, 구독 박스, 멤버십 서비스
- **접근 방식**:
  - 채널별 CAC (Customer Acquisition Cost) 자동 계산
  - 코호트별 LTV 예측 (3개월, 6개월, 12개월)
  - LTV/CAC 비율이 3 이상인 채널만 강조
- **차별점**:
  - 구독 비즈니스는 "첫 구매 ROI"보다 "장기 리텐션 ROI"가 중요
  - Stripe, Chargebee 데이터와 광고 플랫폼 데이터 통합
- **경쟁 우위**: ProfitWell은 결제 분석 중심, 마케팅 ROI 약함
